PEAK

PEAK has a new update called the roots there are a lot of new updates added to the game namely eating all new mushrooms in a single run, using the rescue hook on a friend from 30 meters up, cooking and eating a mandrake and, curing a zombie bite

mushrooms

getting all mushrooms in a single run is surprizingly difficult there are four types a yellow, purple. blue, and red one. you need to get and eat all in a single run which is harder that the original mycology badge because they can only spawn in one biome. I recommend turning it to tender foot so the fog does not catch up to you so you have as much time to find the mushrooms as possible

rescue hook

this one may take some setting up but it is the easiest of the four. get a friend at the bottom of the moutain climb a while up with the rescue hook and shoot them up to you.

mandrake

this one takes a lot of preparation you need a portable stove and a mandrake in the same spot and to do this fast or else you will pass out from the screaming but set the stove down and then cook and eat the mandrake

zombie bite

this requires setting up I recommend getting a medicinal root from the shore and then when you get to the roots let the zombie bite you then eat the root before you pass out
